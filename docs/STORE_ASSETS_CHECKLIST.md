# Store Assets Checklist

This is the capture and art plan for the first Microsoft Store listing.

## Minimum Microsoft Requirements

- At least `1` desktop screenshot is required
- `4+` desktop screenshots are recommended
- Desktop screenshots must be `.png`
- Desktop screenshots must be at least `1366 x 768`
- `1:1` box art is required
- `2:3` poster art is recommended

## Recommended Desktop Screenshot Set

Capture at least these six screenshots:

1. Two frameless sticky notes on the Windows desktop with readable markdown content.
2. Right-click menu open on a note, showing note switching and note actions.
3. A note being edited while the tray icon is visible, showing the app can stay in the background.
4. Storage folder open in Explorer, showing markdown note files in `Simple Sticky Notes`.
5. A note reopened after restart or relaunch, showing window restore.
6. The app alongside Obsidian or a markdown editor, demonstrating file-based interoperability.

## Capture Rules

- Use a clean Windows desktop background with no unrelated personal content visible.
- Keep note text realistic and readable.
- Avoid placeholder lorem ipsum.
- Use note examples that demonstrate markdown and real workflow value.
- Do not show debugging overlays, development terminals, or raw test data.
- Make sure tray and desktop visuals are sharp at the final capture resolution.

## Art Deliverables

- Required: `1:1` box art derived from the current app icon
- Recommended: `2:3` poster art with one or two sticky notes on a clean desktop scene
- Optional later: short trailer showing create, edit, hide, restore, and file-based note storage

## Visual Direction

- Lead with the sticky-note windows, not installer dialogs
- Emphasize desktop presence and readable note content
- Show that notes are local files, not a cloud-only workspace
- Keep the tone practical and desktop-native rather than playful or cartoonish

## Capture Checklist

- hide personal files, notifications, and account names
- use consistent note colors across screenshots
- keep text size readable at Store thumbnail scale
- export PNG files at desktop resolution
- review every image at 100 percent before upload
