# Privacy Policy Draft

Last updated: `2026-04-22`

This draft is intended to become the public privacy policy URL used for the Microsoft Store listing of `Simple Sticky Notes`.

## Summary

`Simple Sticky Notes` is a local-first Windows desktop app. It stores note content and note state in files on your device. It is designed to work with a folder you choose, including an Obsidian-compatible note folder if you want one.

## What The App Accesses

The app may access:

- note content that you create in the app
- note metadata such as window position, size, color, and open or hidden state
- local folders and files needed to read and write your notes
- a configured Obsidian vault folder when you choose to store notes there

## What The App Does Not Require

The app does not require:

- a user account
- a cloud account
- a subscription
- a built-in remote sync service operated by the publisher

## How Data Is Stored

The app stores data locally on your device:

- note bodies are stored as markdown files
- note state is stored as local sidecar metadata files

The default storage location may be a detected Obsidian vault folder or another local folder selected by the user.

## Data Sharing

The app itself is not designed to send your note content to the publisher's servers.

The app may open external programs or links only when you request that behavior, such as:

- opening the storage folder in File Explorer
- opening a note in Notepad
- opening a note in Obsidian

If you choose to store note files in a third-party synced folder, cloud drive, or Obsidian workflow, the privacy behavior of those separate products is controlled by their own terms and policies, not by this app.

## Analytics And Telemetry

At this time, `Simple Sticky Notes` does not intentionally include first-party analytics, advertising SDKs, or account-based tracking operated by the publisher.

## Uninstall Behavior

Uninstalling the app is intended to remove the application itself. It is not intended to automatically delete the user-created note files stored in your chosen folder.

## Your Choices

You control:

- where note files are stored
- whether the app uses an Obsidian-compatible folder
- whether notes are opened in other local apps
- whether the app remains in the system tray
