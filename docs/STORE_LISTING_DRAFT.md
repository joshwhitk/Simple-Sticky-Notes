## Short Description

`Desktop sticky notes for Windows that store each note as a normal markdown file, with local sidecar metadata for window state and Obsidian-compatible file storage.`

## Full Description

`Simple Sticky Notes is a standalone Windows sticky-note app built for people who want real desktop notes without locking their content into a proprietary notes database.

Each note is stored as a normal markdown file, so your note bodies remain readable outside the app. Window position, size, and other sticky-note state are stored separately in local sidecar metadata. The app works especially well with Obsidian-compatible storage, but it can also use another local folder if you prefer.

Use it when you want fast desktop notes, startup restore, tray-based background behavior, and direct access to your files.

Key benefits:
- standalone sticky-note windows for Windows desktop use
- markdown note files you can read and edit directly
- local file ownership instead of cloud-only storage
- Obsidian-compatible folder layout
- startup restore and tray-based background runtime
- note reopening, note switching, and direct file access from the note menu`

## App Features

Each line below fits the current Partner Center per-feature length limit.

- `Standalone sticky-note windows that stay independent from Obsidian at runtime.`
- `Stores each note body as a normal markdown file instead of a hidden app database.`
- `Keeps note window state in sidecar metadata so notes reopen where you left them.`
- `Supports Obsidian-compatible storage layouts for users who manage notes as files.`
- `Restores open notes after login and keeps hidden notes hidden until reopened.`
- `Runs in the system tray so notes can be hidden without shutting down the app.`
- `Reacts to external markdown edits so open notes can reflect file changes.`
- `Supports quick note creation from desktop and startup integration.`
